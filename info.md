ช่วยค้นหาหน่อยตอนนี้มีเว็บเบราว์เซอร์ที่เป็นโอเพ่นซอสหรือไม่และมีเว็บเบาเซอร์ที่เหมาะกับการเขียนเทสต์เว็บไซต์หรือไม่ที่ออโตเมติกเทสต์ได้และง่ายต่อการเขียนเทสต์สามารถเรคคอร์ดการเทสบนเว็บและสามารถแปลเป็นสคริปต์เพื่อเทสต์ได้ในอนาคตมีหรือไม่ช่วยค้นหาหน่อย

มีครับ! ในปัจจุบันมีทั้งเว็บเบราว์เซอร์ที่เป็นโอเพ่นซอร์ส (Open Source) และมีเครื่องมือสำหรับการทดสอบเว็บไซต์ (Automated Testing) ที่สามารถ "บันทึก (Record) การกระทำบนหน้าเว็บแล้วแปลงเป็นสคริปต์อัตโนมัติ" ซึ่งตอบโจทย์ที่คุณกำลังมองหาเลยครับ

นี่คือข้อมูลที่แบ่งตามความต้องการของคุณครับ:

1. เว็บเบราว์เซอร์ที่เป็น Open Source
สำหรับเบราว์เซอร์ที่เป็นโอเพ่นซอร์สเต็มรูปแบบ และเป็นที่นิยมนำมาใช้เป็นฐานในการทำ Automation ได้แก่:

Chromium: เป็นโปรเจกต์ต้นแบบของ Google Chrome, Microsoft Edge และ Brave เป็นเบราว์เซอร์ที่นักพัฒนานิยมใช้ทำ Automated Test มากที่สุด เพราะมีสถาปัตยกรรมที่รองรับการควบคุมผ่านโค้ดได้ดีเยี่ยม

Firefox (Mozilla): ใช้เอนจิ้น Gecko ซึ่งเป็นโอเพ่นซอร์ส 100% และรองรับการทำ Automated Test ผ่านมาตรฐาน WebDriver อย่างสมบูรณ์

Brave / Firefox Developer Edition: เป็นเบราว์เซอร์ทางเลือกที่สร้างต่อยอดมาจากโปรเจกต์โอเพ่นซอร์สข้างต้น

2. เครื่องมือที่ Record หน้าเว็บแล้วแปลงเป็น Test Script ได้เลย
สำหรับโจทย์ที่คุณต้องการคือ "คลิกๆ บนเว็บ แล้วให้มันเจนโค้ดเทสต์ออกมา" มีเครื่องมือระดับอุตสาหกรรมที่ทำได้ยอดเยี่ยมและฟรีดังนี้ครับ:

1. Playwright Codegen (แนะนำที่สุด)
Playwright เป็นเฟรมเวิร์กจาก Microsoft ที่มาแรงมากในตอนนี้ มันมีฟีเจอร์ที่เรียกว่า Test Generator

วิธีทำงาน: คุณแค่รันคำสั่ง (เช่น npx playwright codegen) ระบบจะเปิดเบราว์เซอร์ (Chromium/Firefox/WebKit) ขึ้นมาให้ 1 หน้าต่าง และหน้าต่างเขียนโค้ดอีก 1 หน้าต่าง เมื่อคุณเอาเมาส์ไปคลิก พิมพ์ข้อความ หรือเปลี่ยนหน้า มันจะเขียนสคริปต์ (เลือกภาษาได้ทั้ง JavaScript, TypeScript, Python, C#) ให้แบบ Real-time เลย

ข้อดี: สคริปต์ที่ได้สะอาด นำไปใช้ทำ Automated Test ต่อได้ทันที และตัว Playwright เองเสถียรมาก

2. Chrome DevTools Recorder (มีอยู่ใน Chrome/Chromium เลย)
ฟีเจอร์นี้ฝังมาในเบราว์เซอร์ Chrome และ Chromium อยู่แล้ว ไม่ต้องติดตั้งโปรแกรมเพิ่ม

วิธีทำงาน: กด F12 เพื่อเปิด Developer Tools > หาแท็บที่ชื่อว่า "Recorder" > กดปุ่ม Start recording > ทำการคลิกหรือใช้งานเว็บตามปกติ > กด Stop

ข้อดี: สามารถกด Export ออกมาเป็นสคริปต์ของ Puppeteer, Playwright หรือแม้แต่สคริปต์สำหรับรันผ่าน Node.js ได้โดยตรง สะดวกมากสำหรับการเริ่มต้น

3. Cypress Studio
Cypress เป็นเฟรมเวิร์กเทสต์ฝั่ง Frontend ที่ได้รับความนิยมมาก มีฟีเจอร์ที่ชื่อว่า Cypress Studio (ต้องเปิดตั้งค่า experimentalStudio: true ในคอนฟิก)

วิธีทำงาน: ให้คุณกดคลิกบน UI ของ Cypress เพื่อให้มันเรคคอร์ดการกระทำต่างๆ และเพิ่มคำสั่งลงไปในไฟล์ Test script ปัจจุบันให้โดยอัตโนมัติ

4. Selenium IDE
เป็น Extension คลาสสิกที่ติดตั้งได้ทั้งบน Chrome และ Firefox

วิธีทำงาน: กดปุ่ม Record บน Extension แล้วใช้งานเว็บตามปกติ มันจะบันทึกเป็นสเต็ปการเทสต์ และสามารถ Export โค้ดออกมาเป็นภาษาต่างๆ (Python, Java, JavaScript) ได้

คำแนะนำเพิ่มเติม
หากคุณต้องการนำสคริปต์เหล่านี้ไปใช้ต่อยอดในโปรเจกต์ระบบอัตโนมัติ หรือใช้งานร่วมกับการสั่งงานแบบ CLI แนะนำให้ลองเล่น Playwright หรือ Chrome DevTools Recorder ดูครับ เพราะโค้ดที่ Export ออกมา (โดยเฉพาะ JavaScript/TypeScript หรือ Python) สามารถนำไปประยุกต์ใช้งานหรือปรับแต่งเพิ่มเติมได้ง่ายที่สุดครับ
